Elden Ring

Description

Elden Ring est un jeu de rôle d'action qui se déroule dans le royaume de l'Entre-Terre déchiré après la fracture du Cercle d'Elden. Le joueur incarne un Sans-Éclat, un exilé du royaume, rappelé sur le continent pour restaurer le Cercle d'Elden et devenir le Seigneur d'Elden.
